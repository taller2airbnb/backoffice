Sign In

To sign in, enter email address and password of an admin account, and press the sign in button. If not a user, click on "Don't have an account? Sign Up" to register.

 

Users: Create User

This screen allows you to create a new BookBNB user. All fields are required to create the user. The user types that can be selected are Host or Guest.

 

Users: User List

This screen lists all current users in the service, including information like name, alias, main, national Id, key in the database, and whether they're blocked or not.

The screen also allows admins to block users and forbid their use of both the backoffice and mobile app by pressing the block button. This can be reverted.

 

Users: Transactions

This screen lists the transactions performed by all users on the platform. Includes information such as booker (payer), owner (recipient), type of operation, over which post the transaction was performed, amount of Ethereum deposited, and date in which the transaction took place.

 

Postings

This screen lists all the postings on the platform: their names, content, owner, date of creation, duration (starting/ending date), location and coordinates, maximum number of guests, price per day in Ethereum, and block status.

The screen also allows the admin to block postings and preventing them from showing up on the mobile app by pressing the block button. This can be reverted.

 

Servers

The Servers screen lists all current servers, their API key token and current status.

Servers can be blocked (pressing the block button on the respective row). We advise against blocking the backoffice server, since it is not easily reversible for obvious reasons. New servers can be added by pressing the "add server" button, entering a new name and pressing "create".

 

Covid Warn Setings

BookBNB makes use of an external service to inform users of the state of the ongoing COVID-19 pandemic in the location they plan on visiting. The parameters for this service can be changed by an administrator in the Covid Warn > Covid Settings screen.

The parameters that can be modified are the following:

  • Fatality Rate: fatality rate (0-100) used to measure high Covid rate.
  • Fatality Rate Variation: Fatality Rate variation within a given period.
  • Jump Days: lookback period to calculate fatality rate.
  • Number Days Window Delta: number of days used to calculate case delta.
  • Threshold Slope Variation: amount of variation between daily case curve used.
  • Total Jumps: lookbacks.
